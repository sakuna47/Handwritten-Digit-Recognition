# Handwritten Digit Recognition using CNN

## Overview
This project uses a Convolutional Neural Network (CNN) to recognize handwritten digits from the MNIST dataset.

## Requirements
- Python 3.x
- TensorFlow
- Keras
- NumPy
- Matplotlib
- Scikit-learn
